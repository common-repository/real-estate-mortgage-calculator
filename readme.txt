== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin
2. Unzip real-estate-mortgage-calculator.zip
3. Upload the entire 'real-estate-mortgage-calculator' folder to your plugins directory
4. Go to the Plugins page in your WordPress Administration area and click to 'Activate' "Real Estate Mortgage Calculator" on WordPress.
5. Go to Appearance->widgets and add the "Real Estate Mortgage Calculator" sidebar widget to your sidebar
